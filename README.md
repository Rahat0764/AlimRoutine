A beautiful, sleek, and highly responsive exam routine tracker for the **Bangladesh Madrasah Education Board (Alim 2026)**. Available as a Web App and a Native Android App with interactive home-screen widgets.

## ✨ Features
- **🌐 Cross-Platform:** Works flawlessly on modern web browsers and Android devices.
- **📱 Native Android Widgets:** Includes elegant Medium and Large widgets for the home screen to keep track without opening the app.
- **⏳ Live Countdown:** Real-time countdown timer to the very next exam.
- **🌓 Theme Support:** Beautifully crafted Dark and Light modes.
- **🔄 Group Switching:** Seamlessly switch between Science and Arts groups (with optional subjects) directly from the app or widget.

## 🚀 Installation & Usage

### For Android Users (APK)
1. Go to the [Releases](https://github.com/Rahat0764/AlimRoutine/releases) page.
2. Download the latest `AlimRoutine_V1.apk`.
3. Install the application (Make sure "Install from Unknown Sources" is enabled in your settings).
4. Long-press your home screen to add the interactive widgets!

### For Web Users
Simply visit the live website (PWA supported): *[alim.pro.bd]*

## 🛠️ Tech Stack
- **Web Interface:** HTML5, Tailwind CSS, Vanilla JavaScript
- **Android App:** Java, Android SDK, AppWidgets
- **Built With:** Compiled entirely on mobile using AIDE (Android IDE) - *No PC required!*

## 💻 Build from Source
1. Clone this repository:
   ```bash
   git clone https://github.com/Rahat0764/AlimRoutine.git
   ```
2. Open the project in **AIDE** (on Android) or **Android Studio** (on PC).
3. Build and run the project directly on your device.
